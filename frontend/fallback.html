Diagnóstico de Tela Branca

Essa página é uma alternativa de fallback quando o aplicativo React não carrega corretamente.

Possíveis causas:

  • Erro de JavaScript no aplicativo React
  • Incompatibilidade de dependências
  • Problema com bibliotecas externas (MUI, react-big-calendar)
  • Erro na inicialização do contexto de autenticação

Soluções:

  • Verifique o console do navegador para erros específicos
  • Tente acessar uma página específica diretamente: /login
  • Limpe o cache do navegador e os dados locais
